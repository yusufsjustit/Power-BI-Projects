# 📊 Power BI Project – Data Loading, Transformation & Dashboard Design
*A practical Power BI project demonstrating my skills in data modelling, report building, and dashboard creation.*

## 🔍 Overview  
This repository contains my Power BI work from Week 2 of the Data Technician programme.  
The project focuses on connecting to data sources, transforming data with Power Query, building visual reports, and creating dashboards using Power BI Desktop.

---

## 🚀 Skills Demonstrated  

### ✔️ Getting & Transforming Data (Power Query)  
- Importing data from multiple sources  
- Applying transformations:
  - Cleaning & shaping data  
  - Removing/renaming columns  
  - Data type corrections  
  - Filtering rows  
  - Merging & appending tables  
- Loading transformed data into the Power BI data model  

### ✔️ Data Modelling  
- Understanding relationships between tables  
- Building a structured data model suitable for reporting  
- Creating clean and efficient dimensional structures  

### ✔️ Report Design  
- Creating visuals such as:
  - Bar charts  
  - Line charts  
  - Cards  
  - Tables & matrices  
  - Filters and slicers  
- Applying formatting for readability  
- Designing report layouts to highlight insights  
- Using titles, labels, colour themes, and consistent styling  

### ✔️ Dashboard Creation  
- Building interactive dashboards in Power BI  
- Combining multiple report visuals  
- Setting up navigation and user‑friendly layouts  
- Creating insight‑driven views for business users
